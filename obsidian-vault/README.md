# Phase One Visuals — Knowledge Vault

This Obsidian vault archives all findings, insights, and data outputs generated by Graphify and OpenBrain for the Phase One Visuals website project.

## Structure

| Folder | Purpose |
|--------|---------|
| `graphify-outputs/` | Knowledge graphs, reports, and visualizations from Graphify |
| `openbrain-findings/` | Long-term memory snapshots and search results from OpenBrain |
| `codebase-insights/` | Human-written notes, architecture decisions, and project context |
| `workflow-logs/` | Timestamped logs of Graphify builds and OpenBrain queries |
| `templates/` | Reusable templates for consistent note-taking |

## Workflow

1. **Graphify** builds a knowledge graph from the codebase (`graphify .`)
2. Output goes to `graphify-out/` → archived in `graphify-outputs/` with date stamps
3. **OpenBrain** performs semantic search and stores insights → archived in `openbrain-findings/`
4. Manual insights and architectural notes go in `codebase-insights/`
5. All workflow logs are saved in `workflow-logs/` with correlation IDs
