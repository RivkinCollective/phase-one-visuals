# Phase One Visuals

This project has been migrated to a personal GitHub account for full ownership and control.

## Quick Start
- Open `index.html` in a browser to view the site
- Open `admin/index.html` for the admin panel
- Firestore backend requires Firebase credentials in `assets/js/firebase-config.js`

## AI-Powered Knowledge Management
- **Graphify** (`/graphify .`) — Build a knowledge graph of the codebase
- **OpenBrain** — Semantic search and long-term memory
- **Obsidian Vault** (`obsidian-vault/`) — Archive findings and insights
